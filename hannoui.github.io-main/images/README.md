# Image Files Guide

Place your images in this folder with these filenames:

## Profile Picture
- **profile.jpg** - Your professional headshot or profile picture (220x220 px recommended)

## Experience/Engagement Images
- **experience1.jpg** - Academic Bureau Member photo
- **experience2.jpg** - Outdoor Recreation photo
- **experience3.jpg** - Event Facilitator photo
- **experience4.jpg** - Class Representative photo
- **experience5.jpg** - Protocol Committee Member photo

All experience images should be around 280x180 px or similar aspect ratio for optimal display.

## Certificate Images
- **cert-outdoor.jpg** - Outdoor Recreation Certificate
- **cert-duke.jpg** - Duke of Edinburgh Award Certificate

Certificate images should be around 280x180 px or similar aspect ratio.

## Image Format Tips
- Use JPG, PNG, or WebP formats
- Keep file sizes under 500KB per image for fast loading
- Recommended dimensions:
  - Profile picture: 220x220 px (or square aspect ratio)
  - Experience/Certificate: 280x180 px or 400x300 px
